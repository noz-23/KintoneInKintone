# KintoneInKintone

## Kintoneの詳細表示に別の詳細表示プラグイン

## 1.概要

Kintoneの詳細表示をリンクで指定した別のKintone詳細表示するプラグインです。

別のKintoneの詳細のコメント表示等の表示内容は変更出来ます。
